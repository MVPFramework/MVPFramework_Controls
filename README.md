# MVPControls
为MVPFramework提供一些winform平台下的组件。当然，不使用MVPFramework，也可以使用这些组件。

## 组件列表
|       组件       |  状态   |
| --------------- | ------- |
| TextButton | success |

### Getting Started
Nuget:
```
nuget Install-Package MVPControls -Version 2020.10.1
```

github  
1. clone 项目
```
git clone git@github.com:MVPFramework/MVPFramework_Controls.git
```
2. 编译项目，获取DLL引用即可
